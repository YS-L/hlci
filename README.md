# hlci

A LOLCODE interpreter written in Haskell.

## Compiling

To compile the interpreter using stack, run the following:

```
stack build
stack test
stack install
```

## Usage

```
hlci filename.lol
```

or via stack without installing:

```
stack exec hcli filename.lol
```
